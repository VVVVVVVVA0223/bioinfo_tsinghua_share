# Lesson 1 学习笔记 ### 4 steps of bioinformatics： #### 1. Question Based on biological/medical knowledge #### 2. Information Get biological/medical data #### 3. Analysis Do data clean and feature extraction work #### 4. Modeling Build up probabilistic model and computational algorithm ##### the differences between Algorithm and Model - **Algorithm（算法）** - 用于**解决问题**的**明确**而**有限**的步骤或规则 - 不依赖于数据而**独立存在**的通用工具 - 流程较为固定 - 例：排序算法（Number Sorting Algorithm），动态规划算法（Dynamic Programming Algorithm） ![Algorithm](https://thu-private-qn.yuketang.cn/slide/16001079/cover2286_20250221154443.jpg?e=1740384420&token=IAM-gs8ue1pDIGwtR1CR0Zjdagg7Q2tn5G_1BqTmhmqa:lG4QtEJshc63F8sGQ74VwX9-9cM=) - **Model（模型）** - 用于**描述或预测**事物行为的对数据或者现象的**数据或逻辑抽象**模型 - **依赖数据或现象**来训练或构建产生 - 流程不一定固定，可能需要调整参数或者训练优化 - 例：线性回归模型（Linear Regression Model），神经网络模型（Neural Network Model） ![Model](https://thu-private-qn.yuketang.cn/slide/16001079/cover2285_20250221154442.jpg?e=1740384420&token=IAM-gs8ue1pDIGwtR1CR0Zjdagg7Q2tn5G_1BqTmhmqa:WbNOKYgF70dseBFmp_znbLNiCko=) 课程学习网址：[Getting Started | Bioinformatics Tutorial](https://book.ncrnalab.org/teaching/getting-started) ------ # Bioinformatics课程学习计划 ![](https://thu-private-qn.yuketang.cn/slide/16001079/cover2686_20250221151551.jpg?e=1740385975&token=IAM-gs8ue1pDIGwtR1CR0Zjdagg7Q2tn5G_1BqTmhmqa:_zV_8xHYeo7c61GIXihfKeGU5OY=) - 通过前4周的课程初步掌握Linux系统的使用方法和生物信息学分析技术 - 通过5-16周的课程掌握使用R语言分析处理数据（RNA-seq数据、genome数据等） - 能够将学到的技术用于自己的实验室课题的转录组分析及全基因组关联分析中
