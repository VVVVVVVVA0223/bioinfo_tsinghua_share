the differences between Algorithm and Model
  • Algorithm（算法）
    • 用于解决问题的明确而有限的步骤或规则
    • 不依赖于数据而独立存在的通用工具
    • 流程较为固定
    • 例：排序算法（Number Sorting Algorithm），动态规划算法（Dynamic Programming Algorithm）
  • Model（模型）
    • 用于描述或预测事物行为的对数据或者现象的数据或逻辑抽象模型
    • 依赖数据或现象来训练或构建产生
    • 流程不一定固定，可能需要调整参数或者训练优化
    • 例：线性回归模型（Linear Regression Model），神经网络模型（Neural Network Model）
